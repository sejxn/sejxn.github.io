문제 링크: <a href="https://www.acmicpc.net/problem/1013" target="_blank">https://www.acmicpc.net/problem/1013</a>  

### 풀이  
정규표현식 문제다. 정규표현식은 문자열과 대응되는처음에는 정규표현식을 잘 몰라서 ([1][0][0]+[1]|[0][1])+ 이런식으로 했다.

### 핵심  

### code  
```c++
#include <iostream>
#include <regex>

using namespace std;

int t;
string s;

void Solve()
{
    cin >> t;
    regex re("(100+1+|01)+");
    while (t--) 
    {
        cin >> s;
        if (regex_match(s, re))
            cout << "YES" << '\n';
        else
            cout << "NO" << '\n';
    }
}

int main()
{
    cin.tie(NULL);
    ios::sync_with_stdio(false);
    Solve();
    return 0;
}
```
